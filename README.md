# Sudoku

Simple console Sudoku game in C++ created as a project for a class in the 2nd year.


## Solution of Sudoku

Half a dozen different games from .txt file
Used .txt file as a source of solved Sudoku games each presented in a line of numbers

## Features

- Two dimensional array
- 3 levels (EASY, MEDIUM, HARD) of Sudoku - count of shown numbers at the beginning of the game depends on the picked level
- 2 outcomes of false entry: 
  - coloring the wrong entry number red
  - lives count (commented in the code)
- Possible movement by arrows or AWSD/awsd
